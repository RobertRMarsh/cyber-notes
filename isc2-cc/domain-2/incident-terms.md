## Notes
### Breach
- unauthorized access or infiltration of a system where someone accesses PII to either acquire or disclose it.
- NIST SP 800-53 Rev 5
### Event
- observable occurrence in a network or system.
- NIST SP 800-61 Rev 2
### Exploit
- an attack that exploits a system vulnerability.
### Incident
- an event that can or does jeopardize that CIA triad of an asset.
### Intrusion
- event(s) that constitute a deliberate security incident where one attempts to gain unauthorized access to info/system.
- IETF RFC 4949 Ver 2
### Threat
- event with potential negative impact to org.
- NIST SP 800-30 Rev 1
### Vulnerability
- weakness in a system.
- NIST SP 800-30 Rev 1
### Zero Day
- unknown system vulnerability that can avoid detection due to not fitting recognized attack signatures/patterns.
