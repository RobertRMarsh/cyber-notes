## Notes
### Data Life Cycle
- the life cycle of data, information and knowledge
- used to map data flows through an org and relevant controls used at each point
#### Data Life Cycle Steps
##### Create
- creating data, info, or knowledge
##### Store
- storing/recording to make data explicit
##### Use
- Using the data, info, or knowledge
- may result in altering the data 
##### Share
- Sharing data with others, via either copy, or by moving to another location
##### Archive
- Archiving data when it is temporarily not needed
##### Destroy
- Destroying data when it is no longer needed
